DefaultDefaults

This plugin, designed for OsiriX workstation administration, allows you to specify OsiriX preferences to be set every time OsiriX is launched.
For example, you can use this plugin on workstations where users might modify the toolbar, so that the next user will find the toolbar the way he expects it to be filled.

What you need

You need a Property List editor in order to be able to edit a plist file and therefore specify your own preferences' default values. Such application is bundled with every Apple Developer Tools installation, and other third party editors are available on the internet.

Usage

Use your Property List editor to modify the plist file located at DefaultDefaults.osirixplugin/Contents/Resources/defaults.plist. A typical way of doing this is to launch OsiriX, set your default settings live, quit and then copy the properties you want from the plist file located at ÷/Library/Preferences/com.rossetantoine.osirix.plist.
Once you're done, deploy your modified DefaultDefaults.osirixplugin to the computers you want to behave as specified by the plugin.
