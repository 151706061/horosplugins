Grid plugin

Description: Burns a white grid into the pixels of all the images of a series (in memory).

Usage:

(1) Draw or select a rectangle ROI of the size and position of a tile of the grid,
(2) choose grid 1,2,3,4 or 5 (the number corresponds to the line width)

The 2D viewer standard button "Revert" reads the pixels from file anew, so that the grid vanishes off.

The last menu title(s) of the plugin create a copy of the modified series, named with the selected title of the menu.
Fill free to modify the default menu title -grid- inside the info.plist of the plugin
